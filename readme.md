📝 Automação de Lançamento de Vendas (RPA)

Este projeto consiste em um script de RPA (Robotic Process Automation) desenvolvido em Python. Ele automatiza o processo de leitura de dados de vendas a partir de um arquivo CSV e o preenchimento automático em um formulário web (Google Forms).

O tema do projeto foi inspirado na minha experiência atual no setor comercial de peças agrícolas, simulando a digitalização de registros de vendas de componentes e máquinas.

Nota: O formulário e os dados utilizados neste projeto são 100% fictícios e servem apenas para fins de demonstração.

🚀 Funcionalidades

Leitura de Dados: Utiliza a biblioteca Pandas para processar arquivos .csv de forma eficiente.

Automação de GUI: Controla mouse e teclado através do PyAutoGUI para realizar o input dos dados.

Fluxo Inteligente: Gerenciamento de tempo com time.sleep para garantir que o navegador acompanhe a velocidade do script.

🛠️ Tecnologias Utilizadas

Python 3.13.5

Pandas: Manipulação e análise de dados.

PyAutoGUI: Automação de interface gráfica.

Time: Controle de execução e pausas.
